The Maze Portfolio Project

Experience 3D maze navigation with SDL2 raycasting technology.

Learn more Open Maze

Features

Dynamic Floor and Ceiling Rendering

This feature renders the floor and ceiling dynamically, creating a 3D effect. The code calculates the perspective distortion to make the floor and ceiling appear as if they extend into the distance, enhancing the immersive experience. This is achieved by casting rays and determining the correct textures for each pixel row, making the environment appear more realistic.

Real-Time Mini-Map with Player Position

The mini-map provides a top-down view of the entire maze, displaying walls and floors. It also shows the player's current position and direction. This helps players navigate the maze more effectively, providing a clear reference of their location within the game world. The mini-map updates in real-time as the player moves, ensuring accurate and useful navigation assistance.

Real-Time FPS Display

The FPS (Frames Per Second) counter is displayed on the screen to provide real-time performance metrics. This helps in monitoring the game's performance, ensuring it runs smoothly. The FPS counter updates every second, reflecting the current frame rate and allowing developers and players to assess the game's rendering efficiency.

About

The Maze Portfolio Project was inspired by the challenge of creating an engaging and interactive 3D maze navigation experience using SDL2 raycasting technology. This project started as a Portfolio Project for Holberton School.
