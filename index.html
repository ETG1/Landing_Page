About the Project

This project was inspired by my fascination with mazes and 3D environments. I wanted to create a game that provides a visually engaging experience. The project was developed as part of my coursework at ALX, where I aimed to apply my skills in C++ and SDL2 to create an immersive maze game. This project is a portfolio piece demonstrating my abilities in game development.

Check out the GitHub repository for more details.
